Webapp do meninas.comp
=============

Web application do meninas.comp.

### Instalação do pacote
```
$ cd meninas.comp
$ bash run.sh
$ flask initdb
```

### Executar
```
$ flask run
```
A aplicação estará ativa no endereço http://localhost:5000/ .

### Iniciar Banco de Dados
```
$ flask initdb
```

### Ambiente

Iniciar
```
$ . env/bin/activate
```

Sair
```
$ deactivate
```

### Gerar Arquivos Públicos Estáticos

CSS
```
$ grunt
```

JS
```
$ npm run babel
```
