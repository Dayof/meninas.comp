Webapp do meninas.comp
=============

Web application do meninas.comp.

### Instalação
```
$ cd meninas.comp
$ . env/bin/activate
$ export MENINASCOMP_SETTINGS=config.py
$ pip install --editable .
$ export FLASK_APP=meninascomp
$ flask initdb
```

### Executar
```
$ flask run
```
A aplicação estará ativa no endereço http://localhost:5000/ .

### Environment

Iniciar
```
$ . env/bin/activate
```

Sair
```
$ deactivate
```

### Testes
```
$ cd meninas.comp/test
$ python tests.py
```

### Ferramentas
- Back-End
	- Python 2.7
	- Flask
- Fron-End
	- HTML 5
	- CSS
	- Sass
	- React JS
- Banco de dados
	- SQL
